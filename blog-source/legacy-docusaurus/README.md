# Jing's Dev Notes

一个基于 Docusaurus 的个人技术博客，用来沉淀 Java、Spring、数据库、中间件和工程实践内容。

## 本地开发

```bash
npm install
npm run start
```

默认本地预览地址：

```text
http://127.0.0.1:3000/
```

## 生产构建

```bash
npm run build
```

构建产物会生成到 `build/` 目录。

## 内容目录

- `blog/`：按时间发布的技术文章。
- `docs/`：成体系的技术专栏。
- `src/pages/`：首页、项目页、关于页等独立页面。
- `static/`：图片、图标等静态资源。
