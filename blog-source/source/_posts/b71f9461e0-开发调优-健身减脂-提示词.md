你是一名资深 Remotion 动画工程师，请在当前空工作区中创建一个横屏科普动画项目，主题为「减脂误区」。
合作交流（SWAN02_）

请创建项目目录：

`work/fitness-fatloss-video`

使用 Remotion 制作一条 **45 秒、1920x1080、30fps** 的动画视频。

主 Composition 配置如下：

* id: `FitnessFatLossMyths`
* width: `1920`
* height: `1080`
* fps: `30`
* durationInFrames: `1350`

视频风格要求：

整体为 **温暖、手绘、纸片动画感**。
背景使用米白纸张质感，可加入轻微噪点。
主色使用番茄红、叶绿、深墨蓝、暖黄色。
图形元素使用手绘线条、简化人体、餐盘、哑铃、月亮、天平、能量流动线条。
字体使用系统无衬线或本地中文字体，标题加粗，字幕清晰。

禁止出现：

* 科技感
* AI 感
* 霓虹风
* 玻璃拟态
* 3D 数字人
* 赛博发光效果
* 明显生成式素材痕迹
* “快速瘦”“局部燃脂”等不严谨表达

视频核心观点：

减脂不是靠局部燃脂或极端少吃，而是长期可持续的热量缺口、力量训练、足够蛋白质和睡眠。

请按以下 5 个连续场景制作：

### 场景 1：0-6 秒，开场问题

画面内容：
一个人反复练腹部动作，旁边出现疑问气泡。
屏幕标题：

「为什么天天练肚子，肚子还是没瘦？」

旁白方向：
很多人以为，只要猛练肚子，腹部脂肪就会优先消失。

动画要求：
人物轻微做卷腹动作，腹部位置出现小问号和轻微晃动的提示框。

### 场景 2：6-15 秒，误区 1：局部减脂不存在

画面内容：
出现简化身体轮廓，身体不同区域有柔和能量点流动。
标题：

「误区 1：局部减脂不存在」

解释重点：
脂肪动员是全身性的，不是练哪里就瘦哪里。

动画要求：
能量点从身体多个区域缓慢流动，不要只集中在腹部。
用箭头或流线表现“全身参与”。

### 场景 3：15-25 秒，误区 2：吃得越少越快瘦

画面内容：
一个天平，一边是很少的食物，一边是低电量的人体。
标题：

「误区 2：吃得越少越快瘦」

解释重点：
极端少吃可能短期体重下降，但会影响执行力、训练质量和长期坚持。

动画要求：
天平轻微倾斜，人体出现低电量图标，动作变慢，表情疲惫。
整体表达要温和，不恐吓。

### 场景 4：25-36 秒，正确做法

画面内容：
分成三组核心图形：餐盘、哑铃、月亮。
标题：

「真正有效的方法」

三组内容：

1. 适度热量缺口
2. 力量训练
3. 足够蛋白质和睡眠

动画要求：
餐盘出现蔬菜、蛋白质、主食的简单图形。
哑铃轻轻上浮或弹入。
月亮和星星缓慢出现。
三组元素依次进入画面，节奏稳定。

### 场景 5：36-45 秒，总结金句

画面内容：
所有元素简化收拢成一个温暖的总结画面。
主文案：

「减脂拼的不是狠，是稳定。」

副文案：

「可持续的热量缺口 + 力量训练 + 蛋白质 + 睡眠」

动画要求：
文字逐字或分段出现，最后停留 2 秒。
画面干净、有记忆点，不要堆满元素。

技术要求：

1. 动画必须使用 Remotion 帧驱动实现。
2. 使用 `useCurrentFrame()`、`useVideoConfig()`、`interpolate()`、`Easing.bezier()`。
3. 使用 `Sequence` 或 `Series` 控制时间线。
4. 每段 `Sequence` 都设置 `premountFor`。
5. 不使用 CSS animation。
6. 不使用 CSS transition。
7. 字幕和旁白稿分离。
8. 字幕使用 JSON 数据，字段遵循 `@remotion/captions` 的 `Caption` 类型：

```ts
type Caption = {
  text: string;
  startMs: number;
  endMs: number;
  timestampMs: number;
  confidence: number;
};
```

9. 旁白稿以文本形式保存在项目中，不生成 AI 配音。
10. 视频内容数据集中管理，建议创建 scenes/captions 常量文件，包含：

    * 场景时间段
    * 屏幕标题
    * 旁白文本
    * 字幕时间戳
    * 动画关键词

请完成：

* 初始化 Remotion 项目
* 注册 `FitnessFatLossMyths` Composition
* 编写完整动画组件
* 编写字幕数据
* 编写旁白稿文本
* 保证 45 秒完整播放
* 保证 TypeScript 无错误

完成后请提供测试方式：

1. 如何启动 Remotion Studio
2. 如何查看 Composition
3. 如何渲染 still 帧：

   * frame=30
   * frame=450
   * frame=1050
4. 如何最终导出横屏视频

验收标准：

* 视频完整时长为 45 秒
* 分辨率为 1920x1080
* 帧率为 30fps
* 字幕不越界，不遮挡主体
* 画面温暖、手绘、纸片动画感
* 没有科技感、AI感、霓虹、玻璃拟态
* 减脂科普表达严谨，不制造焦虑，不承诺快速瘦
